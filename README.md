# Attention Economy Revenue Simulator (POC #45)

This project is a dashboard that simulates how different social media platforms generate revenue based on user attention and ad load.

### Key Updates (May 14, 2026):
* **UI Improvements:** Restored the deep dark theme and professional layout.
* **Interactive Sliders:** Added real-time revenue calculation for YouTube, TikTok, Instagram, and more.
* **Bug Fixes:** Fixed the 'Rules of Hooks' error and improved Tailwind CSS styling.

### Tech Stack:
* Next.js
* Tailwind CSS
* React Hooks
