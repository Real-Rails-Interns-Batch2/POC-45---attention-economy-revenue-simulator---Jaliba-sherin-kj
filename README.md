# POC #45 - Attention Economy Revenue Simulator - Jaliha

This project is a production-style intelligence dashboard developed for the Real Rails Intelligence Library. It simulates the mechanics of the Attention Economy, visualizing how user time and platform impressions translate into revenue generation.

## 🚀 Features
- **Revenue Simulation Engine:** Visualizes the conversion of time/impressions into platform revenue.
- **Data Insights:** Analysis of ad loads, CPM ranges, and creator revenue-share models.
- **Dashboard Layout:** Optimized 70/30 UI layout for clear data visualization.
- **Obsidian DNA Theme:** Modern, high-contrast dark theme (#030712).
- **Control Panels:** Explanatory panels for 'Why this matters' and 'Who controls the rail'.

## 🛠️ Tech Stack
- **Frontend:** Next.js, TypeScript, Tailwind CSS, shadcn/ui.
- **Visualization:** D3.js, Plotly, Apache ECharts, TanStack Table.
- **Backend/ETL:** Python FastAPI, Pandas, DuckDB.
- **Data Sources:** Our World in Data, World Bank Data.

## 💡 About
This dashboard provides actionable insights into the digital attention economy, helping viewers understand the economic drivers behind modern online platforms.
